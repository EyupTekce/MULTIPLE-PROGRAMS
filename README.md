# MULTIPLE-PROGRAMS
An AI-powered Windows desktop application using Google Gemini API for weather, web search, currency exchange, schedule generation, and file format conversion


⚙️ Setup & Configuration
To use this application, you need a valid Google Gemini API Key. Follow these steps to set it up:

Get your free API key from Google AI Studio.

Open the source code in Visual Studio.

Navigate to Form1.cs and locate line 16:

"private string apiKey = "YOUR_API_KEY_HERE";" 

Replace "YOUR_API_KEY_HERE" with your actual API key.

Build and run the project.

Note: Make sure to keep your API key private and do not share your repository if it contains your key.
